Detailed Description

different norm varieties

a real or complex matrix. Unlike "normal" matrices where the matrix rows are adjacent, this matrix class separates the number of columns and the distance between column elements in order to avoid unnecessary copies of sub matrices (which are frequent in householder transformations).

Also, the matrix does no memory allocation but operates on a double * provided in the constructor.

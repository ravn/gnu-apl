Detailed Description

An APL Value (essentially a Shape and a ravel)

An APL value. It consists of a fixed header (rank, shape) and and a ravel (a sequence of cells). If the ravel is short, then it is contained in the value itself; otherwise the value uses a pointer to a loner ravel.

21 /* the help_def() macro defines help texts used in different places.
22  *
23  * It is called with 5 arguments:
24  *
25  * help_def(arity, prim, name, title, descr)
26  *
27  * the arguments are:
28  *
29  * arity: the arity of the primitive, encoded as follows:
30  * 0: niladic function
31  * 1: monadic function
32  * 2: dyadic function
33  * -1: monadic operator taking one argument
34  * -2: monadic operator taking two arguments
35  * -3: dyadic operator taking one argument
36  * -4: dyadic operator taking two arguments
37  * -5: quasi-dyadic operator (outer product)
38  *
39  * prim: the APL symbol for the primitive
40  *
41  * name: the official name of the primitive
42  *
43  * title: a short description of the primitive
44  *
45  * descr: a longer description what the primitive computes,
46  * possibly including a short example
47  */

24 /* the help_def() macro defines help texts used in different places.
25  *
26  * It is called with 5 arguments:
27  *
28  * help_def(arity, prim, name, title, descr)
29  *
30  * the arguments are:
31  *
32  * arity: the arity of the primitive, encoded as follows:
33  * 0: niladic function
34  * 1: monadic function
35  * 2: dyadic function
36  * -1: monadic operator taking one argument
37  * -2: monadic operator taking two arguments
38  * -3: dyadic operator taking one argument
39  * -4: dyadic operator taking two arguments
40  * -5: quasi-dyadic operator (outer product)
41  *
42  * prim: the APL symbol for the primitive
43  *
44  * name: the official name of the primitive
45  *
46  * title: a short description of the primitive
47  *
48  * descr: a longer description what the primitive computes,
49  * possibly including a short example
50  */

Detailed Description

an iterator for arrays with permnuted axes

An iterator counting 0, 1, 2, ... ⍴,shape but with permuted axes The permutation is given as a Shape. If perm = 0, 1, 2, ... then PermutedArrayIterator is the same as ArrayIterator.
